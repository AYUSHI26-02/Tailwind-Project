# Sipzo Coffee Landing Page ☕️

Sipzo is a modern, responsive **coffee brand landing page** built using **Tailwind CSS**.  
The project showcases a visually appealing hero section, interactive product display, and smooth UI animations inspired by modern coffee brand websites.

The landing page highlights different coffee flavors with dynamic background color changes and image transitions.

## Overview

The **Sipzo Landing Page** is designed using **Tailwind CSS utility classes** to create a clean, modern, and fully responsive layout.  
It focuses on branding, product presentation, and interactive UI effects using minimal custom CSS and JavaScript.

This project is suitable for:
- Tailwind CSS practice  
- UI/UX design improvement  
- Coffee or beverage brand templates  
- Portfolio showcase  

## Features

- Fully responsive design (mobile, tablet, desktop)  
- Modern hero section with branding  
- Interactive coffee image switcher  
- Dynamic background color change  
- Smooth image transition animations  
- Thumbnail-based product selection  
- Clean navigation bar  
- Social media icons section  

## Technologies Used

- HTML5  
- Tailwind CSS  
- JavaScript (ES6)  
- Font Awesome (icons)
