## Overview

The **Global Recipe Explorer** uses the **MealDB API** to fetch recipes dynamically based on:
- Selected country (area)
- User search input (meal name)

The project is designed to practice:
- API integration using `fetch`
- Dynamic DOM manipulation
- Responsive UI with Tailwind CSS
- Clean and readable frontend code

This project is suitable for:
- Frontend practice  
- API-based projects  
- Portfolio showcase

## How It Works

- On page load, country names are fetched from the MealDB API  
- Selecting a country displays meals from that region  
- Searching by name fetches detailed recipes  
- Recipe cards are generated dynamically  
- Instructions can be expanded or collapsed using JavaScript
