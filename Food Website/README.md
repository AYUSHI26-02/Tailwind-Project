## Overview

This Responsive Burger Website represents a complete restaurant landing page with sections such as **Home, About, Menu, Reviews, and Contact**.

It includes:
- Dark & Light mode
- Animated sections
- Category-based menu filtering
- Customer review slider
- Mobile-first responsive navigation

The UI is modern, smooth, and optimized for **mobile, tablet, and desktop** screens.

## Features

- Responsive navigation bar with hamburger menu
- Dark & Light mode toggle (saved in `localStorage`)
- Animated hero section
- Category cards (Food, Snack, Beverage)
- Promotional offer sections
- Menu filtering using tabs (All / Food / Snack / Beverage)
- Customer reviews slider (Swiper.js)
- Scroll-based animations (ScrollReveal.js)
- Scroll-to-top button
- Active navigation link on scroll
- Fully responsive layout
- Clean and modern UI design

## How It Works

1. Navigation menu opens/closes using JavaScript.
2. Dark/Light theme toggles and is stored in `localStorage`.
3. Menu items are filtered using tab-based JavaScript logic.
4. Customer reviews are displayed using Swiper slider.
5. ScrollReveal adds animations on scroll.
6. Header style changes on scroll.
7. Scroll-to-top button appears after scrolling.
